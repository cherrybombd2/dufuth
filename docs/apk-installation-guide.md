# DUFUTH SmartCare APK Installation Guide

This guide explains how to install the DUFUTH SmartCare Android app when it is shared directly as an APK file.

Because this app is not being installed from the Google Play Store, your phone may show a security warning before installation. This is normal for directly shared APK files.

## Before You Start

- Use an Android phone.
- Make sure you received the APK file from an official DUFUTH SmartCare source.
- Do not install APK files from unknown or untrusted people.
- If an older version of DUFUTH SmartCare is already installed, you may need to uninstall it first if the new APK does not update correctly.

## Installation Steps

1. Download or receive the DUFUTH SmartCare APK file on your phone.

2. Open the APK file from your browser downloads, file manager, WhatsApp, email, or wherever it was shared.

3. If your phone says installation from this source is not allowed, tap `Settings`.

4. Enable `Allow from this source` or `Install unknown apps` for the app you used to open the APK, such as Chrome, Files, WhatsApp, or Gmail.

5. Go back to the APK installation screen.

6. Tap `Install`.

7. Wait for the installation to finish.

8. Tap `Open` to launch DUFUTH SmartCare.

## If You See a Security Scan Warning

Android or Google Play Protect may show a warning because the app was not downloaded from the Play Store.

This can look like:

- `For your security, your phone is not allowed to install unknown apps from this source`
- `Play Protect doesn't recognize this app's developer`
- `This app was not installed from Google Play`
- `Scan app before installing`

If you received the APK from an official DUFUTH SmartCare source, you can continue with the installation.

If the phone asks to scan the app, you may allow the scan before continuing.

## After Installation

1. Open DUFUTH SmartCare.

2. Sign in with your account, or create a patient account if you are a new patient.

3. Make sure your internet connection is active so the app can connect to the hospital system.

## Troubleshooting

If the app does not install:

- Check that your phone is Android.
- Make sure you downloaded the full APK file.
- Try opening the APK from the phone's Files app.
- Uninstall any older test version of DUFUTH SmartCare, then install again.

If the app installs but cannot sign in:

- Check your internet connection.
- Confirm that you are using the latest APK shared by the DUFUTH SmartCare team.
- Try closing and reopening the app.

## Safety Note

Only install DUFUTH SmartCare APK files shared through trusted DUFUTH SmartCare channels. Do not install modified or forwarded APK files from unknown sources.
