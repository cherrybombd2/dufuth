# DUFUTH SmartCare

DUFUTH SmartCare is a hospital appointment management app built to connect patients, doctors, and hospital administrators in one system.

## Current Setup

- Flutter project scaffolded for Android
- Material 3 enabled
- Riverpod selected for state management
- go_router selected for navigation

## Planned User Areas

- Patients
- Doctors
- Administrators

## Next Steps

- Connect Firebase Authentication
- Design role-aware onboarding and dashboards
- Add appointment booking, slot management, and notifications
